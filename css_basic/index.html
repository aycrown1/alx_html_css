My first cool project in ALX INTROSWE

ALX Introduction To Software Engineering is a new, lighter and part-time program would work better for many of you juggling full-time jobs and the very demanding ALX SE program that required a minimum of 80 hours a week. And as announced by Julien Barbier

The program is specifically designed for those aspiring to explore coding fundamentals in a flexible, well-paced virtual learning environment.

It's a simplified, streamlined version of our previous 70-hour-per-week training course, focusing on web technologies only. It is now tailored to fit into a more manageable and more flexible schedule of 20 hours per week

The program has been created and will be led and supported by Dr. Ehoneah Obed , Cole Baidoo, Julien Barbier & Guillaume Salva.

A four-month program blending project-based learning with mentorship.

Practical experience with essential tools and web technologies, including HTML, CSS, Python, Javascript and more.

A fully virtual environment, allowing you to learn from anywhere.
